Комментарии по квартире как предмету покупки

  • Критично Не работает тёплый пол (коридор + санузел): это инженерный дефект, не «косметика».
  • Критично Окна: пороки стеклопакетов + створки задевают рамы. Риск затяжных гарантийных работ.
  • Средне Трещины примыканий откосов, дефекты покраски и пола — массовая отделка «на скорость».
  • Средне Сантехника: нет обратного клапана, герметизация ванны некачественная.
  • Позитив Дефекты типовые для новостройки, в большинстве случаев устранимы по гарантии.

Прогресс: 0/10

Рекомендованная стратегия сделки

  1. Подписывать только дефектную ведомость с полным перечнем (уже есть база).
  2. Требовать официальный график устранения + ответственного инженера.
  3. Повторная приёмка только после закрытия инженерии + окон + сантехники.
  4. Если сроки срываются — фиксировать компенсацию/скидку либо удержание части суммы.

Сводная оценка

Риск: средний Квартира годится к покупке, но на текущем состоянии без устранения критичных пунктов входить в финальный этап сделки не рекомендуется.
